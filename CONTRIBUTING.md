# Contributing to AI Survival Guide

Thank you for helping curate this list. The goal is to keep it practical, honest, and well-sourced.

---

## What Belongs Here

A skill belongs in this guide if it meets **all three** of these criteria:

1. **Hard for AI to replicate** — not just "AI isn't good at this yet," but structurally difficult due to embodiment, genuine human connection, or values-based reasoning
2. **Real reason** — you can explain *why* AI struggles, not just assert it
3. **Actionable** — there is a concrete way for a person to start learning it

## What Does NOT Belong

- Skills that are merely "hard for current AI" but will likely be automated in 1–2 years
- Vague soft skills with no concrete learning path
- Skills that require expensive equipment or rare access to learn
- Anything without a source or real-world example

---

## How to Add a Skill

1. Fork the repository
2. Add your skill entry under the appropriate category in `README.md`
3. Follow this format:

```markdown
#### Skill Name / 中文名 (if applicable)

**Why it endures:** [1–3 sentences: structural reason AI cannot replicate this]

**Real example:** [A specific, concrete scenario — not abstract]

**How to learn:**
- [Concrete first step]
- [Resource: book, course, community]
- [Practice method]
```

4. Add any new source to the **Sources** section at the bottom of `README.md`
5. Submit a Pull Request with a short description of why you're adding this skill

---

## How to Improve an Existing Entry

- Fix a broken link → just fix it and submit a PR
- Add a better real-world example → replace, don't append
- Add a learning resource → add it to the bullet list under "How to learn"
- Dispute a claim → open an Issue first with your reasoning

---

## Style Guidelines

- Write in plain English — this guide is for non-technical people too
- Bilingual entries (Chinese + English) are welcome and encouraged
- No hype, no filler — every sentence should earn its place
- No emojis in skill entries (except ⭐ ⚠️ ✅ ❌ where genuinely useful)
- Sources must be linkable and from credible institutions or peer-reviewed research

---

## Scope of "5–10 Years"

We are curating skills that are **structurally** hard for AI — not just currently hard. Ask yourself: "Even if robotics and AI double in capability in 5 years, would this skill still require a human?" If yes, it belongs here.
